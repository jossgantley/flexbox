Introduction

This documentation will overview the basic concepts of CSS Flex Box, a very useful tool in the realm of CSS. Once you understand CSS Flex Box, it can make creating dynamic webpage layouts almost seem effortless, particularly if you need the layout to be responsive.

Things You Should Already Know

Before you get started, there are some things you should already be familiar with.

  • You should have a good knowledge of HTML markup, how to structure elements on a webpage, and how class and id attributes can be assigned to elements.
  • You should also have a basic understanding of CSS itself before attempting to understand the concepts in this documentation. Particularly, how CSS properties are structured and how they interact with the HTML document.

Parent Properties (flex container)

display

The display property is used to define the parent element(container) as a flex container. It can be set to eitherflex or inline-flex.

.container {
display: flex;
}
OR
.container {
display: inline-flex;
}

The difference between the two is often confused with the flex-direction property. Here is a brief description.

  • flex: Child elements are considered block elements, taking up all available space in their parent container.
  • inline-flex: Child elements are considered inline elements, taking up only the space that is occupied by their own contents.

For more information on the on the difference between block elements and inline elements, click here.

flex-direction

The flex-direction property defines the axis and direction in which the flex items are to be placed.

.container {
flex-direction: row;
}
.container {
flex-direction: row-reverse;
}
.container {
flex-direction: column;
}
.container {
flex-direction: column-reverse;
}
  • row(default): Places items from left to right on the horizontal axis.
  • row-reverse: Places items from right to left on the horizontal axis.
  • column: Places items from top to bottom on vertical axis.
  • column-reverse: Places items from bottom to top on vertical axis.

flex-wrap

The flex-wrap property decides whether all the flex items should be placed on one row/column or if they should wrap onto the next column. By default items will not wrap onto the next row/column.

.container {
flex-direction: nowrap;
}
.container {
flex-direction: wrap;
}
.container {
flex-wrap: wrap-reverse;
}
  • nowrap(default): Places all items on one row/column.
  • wrap: Allows items to wrap onto an additional row/column, from top to bottom for rows, or left to right for columns.
  • wrap-reverse: Allows items to wrap onto an additional row/column, from bottom to top for rows, or right to left for columns.

flex-flow

The flex-flow property combines the flex-direction and the flex-wrap into one property with the default value being row nowrap.

.container {
flex-flow: column wrap-reverse;
}

justify-content

The justify-content property deals with the spacing around the flex items on a row/column.

.container {
justify-content: flex-start;
}
.container {
justify-content: flex-end;
}
.container {
justify-content: center;
}
.container {
justify-content: space-between;
}
.container {
justify-content: space-around;
}
.container {
justify-content: space-evenly;
}
  • flex-start(default): The flex items are placed towards the start of the flex-direction.
  • flex-end: The flex items are placed towards the end of the flex-direction.
  • center: The flex items are placed towards the center of the main axis.
  • space-between: The flex items are evenly distributed on the main axis with the first and last items at each end of the container.
  • space-around: The flex items are placed on the main axis so that there is an equal amount of space allotted to both sides of each item.
  • space-evenly: The flex items are placed on the main axis so that there is the same amount of space between each item and the start and end points of the main axis.

align-items

The align-items property is similar to the justify-content property except it deals with the spacing on the cross axis on a given line of items.

.container {
align-items: stretch;
}
.container {
align-items: flex-start;
}
.container {
align-items: flex-end;
}
.container {
align-items: center;
}
.container {
align-items: baseline;
}

align-content

The align-content property is similar to the align-items property in that it deals with the spacing on the cross axis, but different in the sense that it only works when there is more than one line of flex items.

.container {
align-content: stretch;
}
.container {
align-content: flex-start;
}
.container {
align-content: flex-end;
}
.container {
align-content: center;
}
.container {
align-content: space-between;
}
.container {
align-content: space-around;
}
.container {
align-content: space-evenly;
}
  • stretch(default): The lines of flex items stretch to take up the remaining space on the cross axis.
  • flex-start: The lines of flex items are placed towards the start of the container on the cross axis.
  • flex-end: The lines of flex items are placed towards the end of the container on the cross axis.
  • center: The lines of flex items are placed towards the center of the cross axis.
  • space-between: The lines of flex items are evenly distributed on the cross axis with the first and last lines of flex items at each end of the container.
  • space-around: The lines of flex items are placed on the cross axis so that there is an equal amount of space allotted to both sides of each line of items.
  • space-evenly: The lines of flex items are placed on the cross axis so that there is the same amount of space between each line of items and the start and end points of the cross axis.

flex-grow

The flex-grow property allows an item to grow in proportion to the other items. If all the items have a flex-grow property set to 1, they will each take up the same space. If you change one of the item's flex grow property to a value of 2, then it will attempt to take up twice as much space as the other items if necessary. The default value for the flex-grow property is 0 and it does not accept negative values.

.item4 {
flex-grow: 2;
}

If you change one of the item's flex shrink property to a value of 3, then it will attempt to take up three times less space as the other items if necessary. flex-basis

The flex-basis property represents the default size of an item before the rest of the space is allotted. It can either be set to a length(eg. 40px, 35% etc.) or it can be set to auto(default) which is based on the width or height value for that item. When set to 0, the space around the item is not accounted for, but if set to auto, the space is allotted based on the item's flex grow property.

.item4 {
flex-basis: 20vw;
}
